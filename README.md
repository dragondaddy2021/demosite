# 胖胖龍爹地的 Dragon 家族 - 一頁式形象網站

> 一頁式商品形象網站切版作品 | Bootstrap 5 + 響應式設計

## 🔗 線上預覽

- **網站**：https://dragondaddy2021.github.io/demosite/
- **Facebook**：https://www.facebook.com/dragondaddy2021/

## 🛠 技術棧

- HTML5 / CSS3
- Bootstrap 5.1
- Vanilla JavaScript
- 響應式設計（桌機 ≥768px / 手機 <768px 雙斷點）

## ✨ 主要特色

- 響應式 navbar（桌機橫向展開 / 手機漢堡選單）
- 活動訊息區塊（桌機 Grid / 手機 Carousel）
- Facebook Page Plugin 嵌入
- 社群圖示 SVG Sprite 最佳化
- 已驗證 Google Search Console

## 🚀 本機預覽

### 方法 1：VS Code Live Server

安裝 Live Server 擴充套件，右鍵 `index.html` → "Open with Live Server"

### 方法 2：Python

```bash
python -m http.server 8000
```

瀏覽器開啟 `http://localhost:8000`
